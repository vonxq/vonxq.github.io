fetch和pull区别

详解git fetch与git pull的区别
git fetch 命令会将远端数据拉取到你的本地仓库，它并不会自动合并。
基本情况下git pull = git fetch + git merge
如果需要有选择的合并git fetch是更好的选择。效果相同时git pull将更为快捷。
