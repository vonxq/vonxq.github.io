markdown基础

发表于 2017-02-14 | 分类于 Tec ， Blog |
字数统计: 422 | 阅读时长 ≈ 2

本文内容转载自Markdown——入门指南，略有删改。
更多语法可参考Markdown 语法说明 (简体中文版)

标题

#号后面添加空格，后面的文字便会具有标题的格式。#号个数代表标题级数，越多越小。

列表

无序列表

*或-或+后面加一个空格可构成无序列表，效果如下：

  • 林俊杰
  • 明道
  • 张继科

有序列表

用1.2.3.后面加空格即可，如下:

  1. 林俊杰
  2. 张继科
  3. 明道

    引用

    用>之后加引用内容即可，如下

    我是引用内容

图片与链接

图片为：![链接名](链接地址)
链接为：[链接名](链接地址)
例子如下：
图片：

链接：
百度

当页跳转

  1. 先定义一个锚(id)
  1. 然后使用markdown的语法

粗体与斜体

Markdown 的粗体和斜体也非常简单，用两个 包含一段文本就是粗体的语法，用一个 包含一段文本就是斜体的语法，不用空格隔开。
如下：
我是粗体
我是斜体

表格

生成效果：(Table前最好换行，不然可能不显示)

默认左对齐 居中 右对齐
col 3 is right-aligned $1600
col 2 is centered $12
zebra stripes are neat $1

代码框

行内代码(inline code)：`\<code>`,效果：<code>

块代码(block code)：代码每一行的前面都加4个空格或一个tab，效果：
code1
code2

Fenced Code Block：第一行和最后一行都是3个 “ ` “，中间的行是代码，如下

```
\<code>
```
效果：

1…1617
