babel使用小结

babel

Babel 把用最新标准编写的 JavaScript 代码向下编译成可以在今天随处可用的版本。 这一过程叫做“源码到源码”编译， 也被称为转换编译。
通俗的说，就是我们可以用ES6, ES7等来编写代码，Babel会把他们统统转为ES5。
