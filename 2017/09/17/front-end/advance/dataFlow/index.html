单向数据绑定和双向数据绑定

详解单向数据流和双向数据流的区别与各自特点

非UI控件只有单向，不存在双向。

Links

知乎——单向数据绑定和双向数据绑定的优缺点，适合什么场景？

单向数据绑定

model到View
导致数据流也是单向的
单向 + inline event(组件钩子函数)实现数据的修改

单向数据流

优点

便于实施统一状态管理(flux、redux)
便于维护

缺点

代码量大，组件通信麻烦些

双向数据绑定

model和view双向绑定
将数据变更的操作隐藏在了框架内部，调用者不会直接感知

优点

编写简单

缺点

复杂，出错不好追溯
