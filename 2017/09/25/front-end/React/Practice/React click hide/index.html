React实现点击其它地方隐藏组件

Posted on 2017-09-25 Edited on 2022-05-30 In Frontend , React

在网上找了一圈，基本都是通过遮罩实现的，但不是那么想用遮罩，没找着想用的方法。看 antd 就实现得不错，想源码里肯定有，于是分析了一下 antd Select 的源码，找到了个 antd 的实现方式，路是绕了些，但看看源码还是能有点收获的。

一点点题外话:

大清早起来发现 antd 的 Dropdown 是有菜单隐藏方式设置的，又是为自己的粗心买了时间帐…好在阅读源码也有点收获，事不过三就好

详细步骤

  1. 设计为 document 绑定 onmousedown 的方法
  1. 设计实现 document 的 onmousedown 事件回调
  1. 设计为 document 解绑的方法，供 component 不可见或被销毁时调用
  1. 当弹出框可见时，为 document 绑定 onClick 方法，didUpdate 中调用 bindOnDocumentClickHide 方法
  2. 当弹出框不可见时，为 document 移除 onClick 方法，didUpdate 中调用 bindOnDocumentClickHide 方法
  3. 当当前组件被 unmount 时,解绑回调
