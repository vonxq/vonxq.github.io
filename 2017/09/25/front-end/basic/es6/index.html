ES6 语法学习笔记

Let 和 const

Let

特性：

  1. 只在所在代码块内有效
  2. 不存在变量提升（变量提升：未声明前有 undefinded 值，且会覆盖外层同名变量）
  3. 暂时性死区，temporal dead zone，简称 TDZ（let 覆盖 var 作用域，导致 var 不可用，且在 let 声明之前也不可用）
  4. 不允许重复声明（包括和 const、var）
    块级作用域
  5. （ES6 可在块级作用域中声明函数，最好声明变量式函数）
  6. ES5 只有全局作用域和函数作用域，没有块级作用域
  7. Do 方法获取块级作用域返回值

Const

声明一个只读常量，必须在声明时赋值，否则报错。

  1. 不存在变量提升，存在暂时性死区(同样会覆盖同名 var 作用域)
  2. 不能重复声明（包括和 let、var）
  3. 实现是通过不能修改内存地址。对象 const 不可控

顶层对象属性

浏览器：window
Node：global
ES5 的顶层对象属性与全局变量等价
let 命令、const 命令、class 命令声明的全局变量，不属于顶层对象的属性

变量的解构赋值

模式匹配

  1. 数组、对象均可
  2. 等式两边都要是可遍历的结构，如 let [x, y, z] = new Set([‘a’, ‘b’, ‘c’]);
  3. 允许指定默认值（惰性求值，当匹配项严格等于 undefiend 时生效）
  4. 对象赋值结构{属性：变量}={属性：值}，等号两边属性同名才能得到对应值，无次序影响。
  5. 数值对象解构赋值时会被转化成对象
  6. Undefined 和 NULL 不能解构赋值
  7. 函数参数也可以解构赋值（eg.:add([x,y]){return x+y}），可以使用默认值（解构失败时使用）
  8. 尽量不要放置圆括号

用途：

  1. 变换变量值
  2. 函数返回多个值
  3. 函数参数定义，可无序可有序
  4. 快速提取 JSON 数据
  5. 函数参数默认值
  6. 遍历 MAP 结构

字符串扩展

  1. Unicode 支持
  2. codePointAT：JavaScript 内部，字符以 UTF-16 的格式储存，每个字符固定为 2 个字节。对于那些需要 4 个字节储存的字符（Unicode 码点大于 0xFFFF 的字符），JavaScript 会认为它们是两个字符。
  3. fromCodePoint
  4. 字符串遍历接口：for(… of …)，可识别大于 0xFFFF 的码点
  5. at 方法（对应 charAt）
  6. normalize()
  7. includes(), startsWith(), endsWith()有、在头、在尾，返回 bool
  8. repeat(n)重复字符 n 次并返回
  9. padStart()，padEnd()，补全字符串
  10. 模板字符串，可嵌入变量(${}引入，js 表达式均可，函数也可)，可多行，`标识
    正则扩展
    数值扩展
  11. Number.isFinite(), Number.isNaN()、Number.isInteger()
  12. Number.parseInt(), Number.parseFloat()
  13. Number.EPSILON:可接受的误差范围，常量
  14. 安全整数和 Number.isSafeInteger()
  15. Math.trunc()去小数取整、Math.sign()判断正负零
  16. …………………………….
  17. 指数运算符，eg：23=8

函数扩展

  1. 默认值
  2. 与解构赋值配合使用
  3. Length 属性，返回未指定默认值的参数个数
  4. 设置默认值有参数单独作用域
  5. Rest 参数，即…变量，只能是最后一个参数
  6. name 属性
  7. 箭头函数
  1. 返回对象得加大括号
  2. 可与赋值解构、rest…配合使用
  3. This 指向固定化，this 对象是其定义时所在的对象的 this，箭头函数嵌套时 this 为外层对象的 this
  4. arguments、super、new.target 均指向外层函数对应变量
  1. 尾调用（Tail Call）优化（严格模式下有效）。函数 f 的最后一步是调用函数 g（return g()），这就叫尾调用。
  2. 尾递归优化（严格模式下有效）
  3. 允许函数参数尾逗号

数组扩展

  1. 扩展（spread）运算符…
  1. 它好比 rest 参数的逆运算，将一个数组转为用逗号分隔的参数序列。
  2. 主要用于函数调用（函数参数中…是将多个参数整合为一个数组，函数调用使用…将一个数组展开为多个参数）
  3. 替代 apply 方法
    应用：
  4. 合并数组
  5. 和解构赋值结合
  6. 字符串转换为数组[…’hello’]即 [ “h”, “e”, “l”, “l”, “o” ]，比 split 精准
  7. Map、Set、Iterator、Generator
  1. Array.from()
    Array.from 方法用于将两类对象转为真正的数组：类似数组的对象（array-like object）和可遍历（iterable）的对象
  2. Array.of 方法用于将一组值，转换为数组。
  3. 数组实例的 copyWithin(target,start,end)
  4. 数组实例的 find() 和 findIndex()
  5. fill(定值)，用于初始化 or 格式化数组
  6. 数组实例的 entries()键值对，keys()键 和 values()值 ，用于遍历数组
  7. Includes()，是否包含
  8. 数组的空位指定为 undefined（尽量避免）

对象扩展

  1. ES6 允许在对象之中，直接写变量。这时，属性名为变量名, 属性值为变量的值。注：简洁写法的属性名总是字符串
  2. Object.assign：Object.assign 方法用于对象的合并，将源对象（source）的所有可枚举属性，复制到目标对象（target）。

Symbol

原始数据类型，标识独一无二的值。
字符串或数字可以作为属性的键，symbol 也可以，它不等同于任何字符串，因而这个以 symbol 为键的属性可以保证不与任何其它属性产生冲突。
原始数据类型（7）：数值（Number）、字符串（String）、Boolean、Object、undefined、Null、Symbol

Set

成员值唯一，类似数组结构。成员类型不定

  1. 遍历：keys()、values()、entries()、forEach()
  2. WeakSet：成员只能是对象

Map

Object：键值对集合，字符串作为键值
Map: 键值对集合,键值类型不限
遍历：同 Set
WeakMap:只接受对象和 null 值作为键名
Proxy
Proxy 实际上重载（overload）了点运算符，即用自己的定义覆盖了语言的原始定义
Promise

Iterator

任意一个对象的 Symbol.iterator 方法，等于该对象的遍历器生成函数，调用该函数会返回该对象的一个遍历器对象（由 next()方法）。
Genereator
yeild，函数分段执行。

生成器是迭代器。所有的生成器都有内建.next()和Symbol.iterator方法的实现。你只须编写循环部分的行为。
作用：实现异步处理

Thunk 函数

编译器的”传名调用”实现，往往是将参数放到一个临时函数之中，再将这个临时函数传入函数体。这个临时函数就叫做 Thunk 函数。
JS 的 Thunk 函数：
JavaScript 语言是传值调用，它的 Thunk 函数含义有所不同。在 JavaScript 语言中，Thunk 函数替换的不是表达式，而是多参数函数，Thunk 函数将多参带回调的函数替换成单参数的版本，使该函数只接受回调函数作为参数。Thunk 函数接收多参数，返回一个只接受回调函数的函数，即返回一开始被替换的函数。
for…of
for…of：遍历的是值
for…in:数组遍历的是 index，对象遍历的是键

  3. 赋值解构？？？（需要实践）（let,const 实现）析构赋值让我们从 Object 或 Array 里取部分数据存为变量。
    Import 和 export 会用到
    export 在导出接口的时候，必须与模块内部的变量具有一一对应的关系{}
    http://imweb.io/topic/582293894067ce9726778be9
  4. Spread Operator…
    函数参数- 默认值、参数打包、数组展开（Default【函数参数可定默认值 x=1 一类】、Rest【获取数组部分项。函数参数…y 是多个值，进入函数被转换为 y 数组】、Spread【…[1,2,3]】）
  5. 箭头函数
  1. 简化了代码形式，默认 return 表达式结果。
  2. 自动绑定语义 this，即定义函数时的 this。
  1. 字符串模板 Template strings
  2. Iterators（迭代器，next）+ for..of
  3. 生成器（Generrators）
    dva 的 effects 是通过 generator 组织的。Generator 返回的是迭代器，通过 yield 关键字实现暂停功能。
  4. Class（语法糖，有 constractor、extends、super 方法）
    语法糖: 没有给计算机语言添加新功能，而只是对人类来说更“甜蜜”的语法。语法糖往往给程序员提供了更实用的编码方式，有益于更好的编码风格，更易读。不过其并没有给语言添加什么新东西。糖在不改变其所在位置的语法结构的前提下，实现了运行时等价。
  5. Modules(import and export)（AMD+CommonJs）
    ES6 的内置模块功能借鉴了 CommonJS 和 AMD 各自的优点：
  1. 具有 CommonJS 的精简语法、唯一导出出口(single exports)和循环依赖(cyclic dependencies)的特点。
  2. 类似 AMD，支持异步加载和可配置的模块加载。
  1. Map + Set + WeakMap + WeakSet
    四种集合类型，WeakMap、WeakSet 作为属性键的对象如果没有别的变量在引用它们，则会被回收释放掉。
  2. 新 API(Math + Number + String + Array + Object APIs)
  3. Proxies（使用代理（Proxy）监听对象的操作，然后可以做一些相应事情。）
  4. Symbols: Symbol 是一种基本类型。Symbol 通过调用 symbol 函数产生，它接收一个可选的名字参数，该函数返回的 symbol 是唯一的。
  5. Promises：Promises 是处理异步操作的对象，使用了 Promise 对象之后可以用一种链式调用的方式来组织代码，让代码更加直观（类似 jQuery 的 deferred 对象）。
