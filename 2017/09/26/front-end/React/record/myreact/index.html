React实战总结

舍不得删就搬到这来了。
学习和使用React刚好一个月了，太久没接触新知识感觉思维蛮僵化的，还好有前辈的现成代码供参考，算是一条捷径了吧，当然一直这样学太浅也不行，但至少入门的时候能快一些。下面对这一个月碰到的问题及常用技巧做个小总结。

学习的过程中参考了很多前辈写的教程，受益匪浅，但从知道到会用还是得有个过程的，我想好好做总结查漏补缺是很关键的一步

Tips：

1.数据fetch之后调用then将result.data保存为本地state中的变量

3.utils/request.js提供request(调用fetch返回数据)、fetch方法（！！！！！！！请求方法需掌握）
