html基础知识

Posted on 2017-02-27 Edited on 2022-05-30 In Frontend

把之前的前端基础知识拆成三部分，方便查阅

HTML 常用标签

标签分类：

块级元素：div p li h1`h6 等，独占一行，若未设置高度则充满父元素

  1. 行内元素：span a strong 等，不能直接设置宽高
  2. 行内块元素：img input 等，可以直接设置宽高
  1. 元数据，自关闭标签，常用属性：

    1.1. charset=”utf-8” /ASCII/ANSI/UNICODE/GBK/GB2312/BIG5
    1.2. name=”” content=””用于描述页面信息
    name 取值：keyword/description…
    1.3. http-equiv=””content=””相当于 http 的文件头作用，它可以向浏览器传回一些有用的信息。
    http-equiv 取值：content-type/refresh/…

  2. input 标签，自关闭,常用属性：(label for=”id”)

    2.1. type=”” text/password/radio/checkbox/file/submit/button/reset/image
    2.2. maxlengh=”数字”
    2.3. readonly=”readonly”
    2.4. disabled=”disable”(非活动状态)
    2.5. name=””(常与 form 或 checkbox 配合使用)
    2.6. value=””
    2.7. checked=”checked”

  3. form 表单，用于收集数据

    3.1. action=””处理用户数据信息的页面
    3.2. method=” “ get(地址栏显式提交)/post(数据后台提交)
    3.3.

    名内容表单分组控件

  4. select
    属性：multiple=”multiple”
    用法；(select)(optgroup label=””)(option selected=”selected”)(/option)(/optgroup)(/select)

  5. textarea 多行文本输入框

6.table

tr、td、caption、thead、tbody、tfoot
常用属性：border/align/width/height/cellspacing/cellpading
td 属性：colspan=”n”,rowspan=”n”合并 n 个单元格

HTML5 新标签

详情可见Html5 新标签解释及用法
1.(embed src=”” hidden=”true/false”)(/embed)

  1. canvas

  2. <marquee>内容</marquee>滚动效果 属性:

    3.1. behaviour:alternate/scroll/slide
    direction:down/up/..
    width、height、bfcolor

  3. 语义化标签 footer、header、article、aside、nav、summary…

  4. audio、vedio…
